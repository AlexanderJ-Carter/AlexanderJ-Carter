# Security Policy

## Reporting a Vulnerability

This is a personal profile repository. If you discover a security issue or vulnerability:

- **Report via**: GitHub Security Advisories (preferred)
- **Alternative contact**: Open a private issue only if the report is not security-sensitive
- **Response time**: I'll acknowledge receipt within 48 hours
- **Disclosure**: Please practice responsible disclosure
- **Scope**: Broken links, rendering glitches, or profile-content mistakes should be reported as normal issues instead of security reports

## Supported Versions

This repository is continuously updated. The main branch always contains the latest version.

Thank you for helping keep this project secure.
